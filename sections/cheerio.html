Cheerio is a fast, flexible, and lean implementation of core jQuery designed specifically for the server. It's the perfect tool for parsing HTML and extracting data in Node.js environments.

What is Cheerio?

Cheerio parses markup and provides an API for traversing/manipulating the resulting data structure. It doesn't interpret the result as a web browser does, but rather parses HTML and allows you to manipulate it with a jQuery-like syntax.

Installation & Setup

// Install Cheerio and Axios for HTTP requests
npm install cheerio axios

Key Functions & Methods

Method Description Example
load(html) Parse HTML and create Cheerio instance const $ = cheerio.load(htmlString)
find(selector) Find descendants of selected elements $('div').find('.class')
each(function) Iterate over a collection of elements $('li').each((i, el) => { ... })
text() Get the text content of elements $('h1').text()
html() Get the HTML content of elements $('div').html()
attr(name) Get the value of an attribute $('img').attr('src')
addClass(className) Add class to selected elements $('div').addClass('new-class')
removeClass(className) Remove class from selected elements $('div').removeClass('old-class')
hasClass(className) Check if element has specific class $('div').hasClass('active')
parent() Get the parent of selected elements $('span').parent()

Working with HTML Elements

Cheerio provides powerful methods for selecting and manipulating HTML elements:

// Different ways to select elements
const $ = cheerio.load(html);

// Select by tag name
const paragraphs = $('p');

// Select by class
const items = $('.item');

// Select by ID
const header = $('#header');

// Select by attribute
const images = $('[src]');

// Complex selectors
const links = $('div.content a.external');

Data Extraction Examples

Example 1: Extracting Product Information

async function scrapeProducts() {
  const response = await axios.get('https://example-store.com/products');
  const $ = cheerio.load(response.data);

  const products = [];
  $('.product').each((index, element) => {
    const product = {
      name: $(element).find('.name').text().trim(),
      price: $(element).find('.price').text().trim(),
      image: $(element).find('img').attr('src')
    };
    products.push(product);
  });

  return products;
}

Example 2: Extracting Table Data

function extractTableData($) {
  const tableData = [];
  $('table tr').each((index, row) => {
    if (index === 0) return; // Skip header row
    
    const rowData = {};
    $(row).find('td').each((colIndex, cell) => {
      rowData[`col${colIndex}`] = $(cell).text().trim();
    });
    tableData.push(rowData);
  });
  return tableData;
}

Advanced Selectors

Selector Example Description
Element $('div') Selects all <div> elements
Class $('.className') Selects all elements with class="className"
ID $('#idName') Selects the element with id="idName"
Attribute $('[name="email"]') Selects elements with name="email"
Multiple $('div, p') Selects all <div> and <p> elements
Child $('div > p') Selects direct child <p> elements of <div>
Descendant $('div p') Selects all <p> elements inside <div>

Best Practices

  • Always handle errors with try-catch blocks
  • Use specific selectors to avoid extracting unwanted data
  • Respect website terms of service and robots.txt
  • Implement rate limiting to avoid overwhelming servers
  • Cache results when possible to reduce repeated requests
